# Hello World - 我的第一篇文章

欢迎来到我的博客！🎉

这是我使用 **GitHub Pages** 搭建的个人网站，所有文章都使用 Markdown 编写，简单高效。

## 为什么选择这种方式？

- **简单**：只需要写 Markdown，不需要任何构建工具
- **免费**：GitHub Pages 提供免费托管
- **灵活**：纯前端方案，可以随意自定义样式

## 如何添加新文章？

1. 在 `posts/` 目录下创建一个 `.md` 文件
2. 在 `posts.json` 中添加一条记录
3. 提交并推送到 GitHub

就这么简单！
