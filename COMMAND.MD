## AI 构建工程要求

### 项目结构
结构如下： 对功能的增加只能在此基础上增加，不能删除，对现有结构也不能调整

```
├── index.html      # 主页（自动加载文章列表）
├── post.html       # 文章详情页（渲染 Markdown）
├── posts.json      # 文章配置文件
├── posts/          # Markdown 文章目录
│   ├── hello-world.md
│   └── about-me.md
└── README.md
```

### 初始化要求
这个是我的用在github上的一个自我介绍的静态网站，帮我自动生成一个静态网页的网站，要求能够美观，我每次增加文档只需要增加markdown文件即可，并且每次能够将增加的markdown文件生成到主页的列表中去。

### 展示要求
1. 文章的详情页的头部样式要和首页的样式对应，包括颜色。
2. 文章的详情页的尾部样式要和首页的样式对应，包括颜色，字体，大小等。
3. GitHub 旁边要带上 GitHub 的图标。
4. 在GitHub 后边增加一个`关于我`的按钮，点击后跳转到`about-me.md`。

### 功能要求
1. 增加根据标签的过滤的功能，可以根据标签过滤文章。
2. 增加根据日期的过滤的功能，可以根据日期过滤文章。
3. 增加根据文章名称搜索的功能，可以根据文章名称搜索文章。
